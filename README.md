# ルナスタ

Phase 1 starter for a Brawl Stars-style web game on Netlify.

## Included
- Login / registration shell
- Auto-login via localStorage placeholder
- Email verification screen placeholder
- Lobby screen
- PWA manifest + service worker
- Netlify redirect config
